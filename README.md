# trainee-journey-python


Welcome to the **Trainee Journey - Python** repository.

This repository is the central workspace for the Python trainee cohort. It contains all weekly practical assignments, coding exercises, mini-projects, and learning resources completed throughout the training program.

## Objectives

* Build strong Python programming skills through hands-on practice.
* Learn and follow industry-standard Git and GitHub workflows.
* Collaborate using feature branches and Pull Requests.
* Write clean, readable, and maintainable code.
* Maintain a record of all practical work completed during the training.

## Repository Structure

```text
.
├── README.md
├── CONTRIBUTING.md
├── .gitignore
├── .github/
│   └── PULL_REQUEST_TEMPLATE.md
├── Week01/
├── Week02/
├── Week03/
├── Week04/
├── ...
└── WeekNN/
```

Each `WeekNN` folder contains the practical exercises and assignments for that week's topics.

## Git Workflow

This repository follows a **Trunk-Based Development** workflow.

* The `main` branch is always stable.
* Never commit directly to `main`.
* Create a new feature branch for every practical.
* Submit all changes through a Pull Request.
* Merge only after review and approval.

## Branch Naming Convention

Create branches using the following format:

```text
week-NN-topic-slug
```

Examples:

```text
week-01-python-basics
week-02-control-flow
week-03-functions
week-04-object-oriented-programming
week-05-file-handling
```

## Commit Message Guidelines

Write clear and meaningful commit messages.

Examples:

```text
Add Week 01 Python basics practical
Implement Week 03 functions exercises
Fix file handling assignment
Refactor OOP examples
```

## Code Standards

* Follow PEP 8 coding standards.
* Use meaningful variable and function names.
* Write modular and reusable code.
* Remove unused code before submitting.
* Test your code before opening a Pull Request.

## Pull Requests

Every Pull Request should include:

* Week number
* Topic name
* Summary of changes
* Testing details
* Additional notes (if required)

## Contributors

This repository is maintained by the Python trainee cohort as part of the training program.
